# Enigma Cracker
C++ based commercial enigma M3 cracker

# Building this project 
This is a Visual Studio 2019 Project, please build it from there.

# Running the application

The application simply runs to crack the following settings.

```
plaintext	  	: ILOVESOFTWAREENGINEERING
ciphertext	  	: BXMEPGJMGMSYILRMXUJQMOMO	
rotor numbers 	: 4 3 1
rotor settings	: E D I
plugboard		: DEFAULT (NOT SUPPORTED)
```
